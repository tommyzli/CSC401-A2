Comments:

My first observation on the BLEU scores is that higher n-values are more likely to have a score of 0.
This makes sense, as considering the quality of the generated translations, it is likely that a
translation can have no trigrams that match one of the references. Almost all of the BLEU scores with
an n-value of 1 have a non-zero score, which also makes sense as each translation should have at least
a few unigrams in common with the references.

Another pattern is that sentences that scored poorly often scored poorly no matter what size of the
training set was used. For example, line 25 received a BLEU score of 0 for each n-value in each
training set size. Lines 6, 10, and 13 have BLEU scores of 0 for n = 2 and n = 3 no matter which
training set size. They only received non-zero BLEU scores for n = 1. 

Lastly, BLEU scores for the smallest training set is noticeably smaller than the others. The average
BLEU score for the smallest training set is the lowest, while the average scores for the other training
set sizes are all relatively similar. Also the smallest training set has the largest amount of BLEU
scores of 0, as can be seen in the table below.
